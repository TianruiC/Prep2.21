The Mean is the sum of the numbers divided by the amount of the numbers we have.

We can use the Reduce method to let the computer calculate the sum of the numbers for us.

The reduce() method executes a function which you provide on each member of the array resulting in single output value.

In order to calculate the Mean of the numbers in the array, we need to calculate the sum first.

We need to create an function to add a particular number to the total variable

The example code for getSum() function will be

function getSum(total, num)
{
return total + num;
}

Then we need to use the getSum() and apply it to the reduce() method for array to let it run for each member in the array to calculate the sum.

The sum can be calculated by the code : Num.reduce(getSum). And the Num represents the number array, it can be replaced by any array name you given.

After finding the sum of the numbers in the array we need to divided it by the amount of the numbers in the array, which can be calculated by the code: Num.length

So the Mean will can be calculated by code: Num.reduce(getSum)/Num.length;
